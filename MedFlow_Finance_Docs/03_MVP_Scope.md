## MVP NÃO é:

* Prontuário eletrônico
* Telemedicina
* App para paciente
* Sistema clínico completo

## MVP É:

* Ferramenta de **faturamento e validação financeira**

### Funcionalidades obrigatórias

1. Cadastro de clínicas (multi-tenant)
2. Upload de arquivos (Excel / CSV / XML)
3. Parser e normalização de dados
4. Validações automáticas
5. Detecção de inconsistências
6. Dashboard financeiro básico
7. Exportação de relatórios
8. Logs e auditoria

## Critério de sucesso do MVP

Se uma clínica disser:

> “Isso me fez recuperar dinheiro que eu perderia”

O MVP cumpriu o papel.
