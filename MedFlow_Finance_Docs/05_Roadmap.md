## Fase 0 — Planejamento (1 semana)

* Definir regras de faturamento
* Mapear layouts de arquivos
* Definir modelo multi-tenant
* Definir métricas

## Fase 1 — Fundação (2 semanas)

* Laravel 11
* Auth (Sanctum)
* Tenancy por clínica
* Estrutura de permissões
* Setup de filas e jobs

## Fase 2 — Core do MVP (3–4 semanas)

* Upload de arquivos
* Parsers (Excel/XML)
* Regras de validação
* Armazenamento normalizado
* Logs de erro

## Fase 3 — Frontend (2 semanas)

* Vue 3 + Vite
* Dashboard financeiro
* Tabelas e filtros
* Exportações

## Fase 4 — Estabilização (1 semana)

* Testes básicos
* Segurança
* Performance
* Deploy

⏱️ **Tempo total estimado:** 8–10 semanas
